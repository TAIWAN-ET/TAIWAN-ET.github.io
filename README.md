## 🚀 How to Deploy on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com/new)
2. Repository name: `TAIWAN-ET.github.io`
3. Make it public
4. Create repository

### Step 2: Push Your Code
```bash
cd ~/projects/webp2026/hw3
git branch -m main
git remote add origin https://github.com/TAIWAN-ET/TAIWAN-ET.github.io.git
git add .
git commit -m "Initial commit - Portfolio website"
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to repository Settings
2. Scroll to "GitHub Pages" section
3. Source should automatically be set to `main` branch
4. Your site will be published at: https://TAIWAN-ET.github.io
